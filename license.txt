git status 查看仓库当前的状态
git diff   查看修改内容
提交修改和提交新文件：1.git add 文件名
	2.在执行第二步git commit之前，再运行git status查看当前仓库状态。   Git commit -m “描述”
git log    命令显示从最近到最远的提交日志	嫌输出信息太多，看的眼花缭乱，可以加上—pretty=oneline
版本回退：git中，用HEAD表示当前版本，上一个版本HEAD^,当然100个 可写成HEAD～100
	git reset —hard HEAD^ //最新版本
	cat readme.txt 查看内容
git reflog 用来记录每一次的命令
